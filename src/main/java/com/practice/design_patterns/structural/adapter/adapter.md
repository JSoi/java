# ADAPTER - 어댑터 패턴

## :paperclip: 정의

메서드 콜이나 행동을 하나의 객체로 표현하는 것

요청의 발신자와 수신자를 분리한다.

실제 오퍼레이션은 "커맨드"라는 객체로 캡슐화 되어 있다.

## 📜 개념

1. Client
   - adaptee 대신 다른 인터페이스 사용
2. Target Interface
   - Client 측에서 호출하는 인터페이스
3. Adapter
   - Target Interface 구현
   - Adaptee 기능을 Target으로 구현
4. Adaptee
   - 기존 클래스에 존재하는 클래스
   - 호환되지 않는 인터페이스를 가짐
